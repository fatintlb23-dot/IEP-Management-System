# IEP Management System

Final Project Individual Education Plan (IEP) Management System

## About

The IEP Management System is a web-based system designed for special needs education centres to manage student information, IEP goals, weekly progress, and generate reports in an organized and efficient way. It replaces manual paper-based record keeping with a centralized digital platform.

## Features

### Admin
- Manage teacher accounts (add, view, delete)
- Approve/Reject parent registrations with email notifications
- View all student records and progress
- Generate overall progress reports
- Print and export reports as PDF

### Teacher
- Manage student profiles (add, edit, view, search)
- Create and manage IEP goals (Academic, Communication, Behaviour, Self-Help)
- Update weekly progress with percentage and notes
- Generate progress reports (Weekly and 6-Month Evaluation)
- Print and export reports as PDF

### Parent
- Register for account (requires admin approval)
- Check registration status
- View child's profile (view-only access)
- View IEP goals and progress
- View progress reports

## Technologies Used

- **Framework**: Laravel 10
- **Frontend**: Blade Templates, HTML5, CSS3, JavaScript
- **Database**: MySQL
- **Server**: Apache (XAMPP)
- **Development Tools**: Visual Studio Code, phpMyAdmin
- **Programming Language**: PHP 8.2
